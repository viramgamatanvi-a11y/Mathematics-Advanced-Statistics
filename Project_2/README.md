# 📊 Probability Analysis Report

## 📘 1. Understanding the Basics

### 🎯 1. What is Probability?
* ➡️ Probability is a measure of the likelihood that a particular event will occur.
* ➡️ It quantifies uncertainty using a number between 0 and 1:
  * ❌ **0** means the event cannot happen
  * ✅ **1** means the event is certain to happen

### 🧩 2. Key Probability Terminology?
* 🧪 **Experiment** = An action with uncertain results *(e.g., tossing a coin)*
* 🎲 **Outcome** = A possible result of an experiment *(e.g., heads)*
* 🌐 **Sample Space** = All possible outcomes *(e.g., {H, T})*
* 🎯 **Event** = One or more outcomes *(e.g., getting a tail)*
* ⭐ **Favorable Outcomes** = Outcomes that match the desired event

### 📊 3. Dataset Probability Event Examples
* 🎓 Probability that a student passes the final exam
* 📅 Probability that a student has attendance greater than 80%
* 💬 Probability that a student participates in group discussion

---

## 🎲 2. Types of Events

### 📍 1. Empirical Probability (Experimental Probability)
* ➡️ Empirical probability is based on real-world data or actual observations from experiments or datasets.

### 🧠 2. Theoretical Probability
* ➡️ Theoretical probability is based on mathematical logic and assumes all outcomes are equally likely.

---

## 📈 3. Random Variable & Probability Distribution

### 🔢 1. Random Variable
* ➡️ A random variable is a numerical quantity whose value depends on the outcome of a random phenomenon or experiment.

### 📉 2. Probability Distribution
* ➡️ A probability distribution is a mathematical function or rule that provides the probability of occurrence of different possible outcomes in an experiment.
* 📌 **Formula:** $$\sum P(X=x) = 1$$

### 📍 3. Mean of a Random Variable
* ➡️ The mean of a random variable, also called the expected value (denoted by $E[X]$), is the long-run average of outcomes weighted by their probabilities.
* 📌 **Formula:** $$E(X) = \sum x \cdot P(x)$$

### 📏 4. Variance of a Random Variable
* ➡️ Variance measures how much the values of a random variable deviate from its mean.

---

## 🟢 4. Venn Diagram in Probability
* ➡️ A Venn diagram is a graphical representation of sets and their relationships. In probability, it helps visualize events, intersections, unions, and complements.

---

## 📋 5. Contingency Tables & Probability Calculation

### 📊 1. Contingency Tables
* ➡️ A contingency table shows the frequency distribution of variables and helps calculate joint, marginal, and conditional probabilities.

### 🤝 2. Joint Probability
* ➡️ Joint probability is the probability that two events happen at the same time.

### 📌 3. Marginal Probability
* ➡️ Marginal probability is the probability of a single event happening, regardless of any other events.

### 🔗 4. Conditional Probability
* ➡️ Conditional probability is the probability of an event given that another event has already occurred.

---

## 🔍 6. Understanding Relationships

### 🔄 1. Conditional Probability
* ➡️ Conditional probability measures the probability of an event happening given another event has occurred.

### ⚖️ 2. Independent Events
* ➡️ Two events A and B are independent if one does not affect the other.

### 🔗 3. Dependent Events
* ➡️ Two events A and B are dependent if one affects the probability of the other.

### 🚫 4. Mutually Exclusive Events
* ➡️ Two events A and B are mutually exclusive if they cannot happen together.

---

## 🧠 7. Bayes Theorem Application
* ➡️ Bayes’ Theorem is a way to update probabilities when you have new evidence.
* ➡️ It tells you how likely an event is given that another event has occurred.
