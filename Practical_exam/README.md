# 📊 Employee Performance Analysis Project

## 📌 Project Overview

This project analyzes employee performance data using **Statistics, Probability, Data Visualization, and Linear Algebra** concepts.

The dataset contains employee information such as:

* 👤 Employee_ID
* 🏢 Department
* 🎂 Age
* 💰 Salary
* 📋 Projects_Completed
* ⏰ Working_Hours
* 📈 Performance_Score
* 🎯 Promotion_Status

The main objective is to explore employee performance patterns and understand the probability of promotion using Python-based analysis.
